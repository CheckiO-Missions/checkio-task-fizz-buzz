The final answer is built from parts: if the string is multiplied by 0, it becomes an empty string "". 

def checkio(n: int) -> str:

    return 'Fizz'*(not n%3)+' '*(not n%15)+'Buzz'*(not n%5) or str(n)

In this solution division by 15 is avoided, but in case num is evenly divisible both by 5 and 3, it means, that it is also evenly divisible by 15. So both strings are present in the list.
