Το "Fizz buzz" είναι ένα παιχνίδι λέξεων που θα χρησιμοποιήσουμε για να διδάξουμε διαίρεση στα ρομπότ.

Σκοπός είναι να γράψεις μια συνάρτηση που θα δέχεται ένα θετικό ακέραιο και θα επιστρέφει:
"Fizz Buzz" αν ο αριθμός διαιρείται με το 3 και με το 5
"Fizz" αν ο αριθμός διαιρείται με το 3
"Buzz" αν ο αριθμός διαιρείται με το 5
Τον αριθμό σε μορφή string σε όλες τις υπόλοιπες περιπτώσεις.

Είσοδος: Ένας ακέραιος αριθμός.

Έξοδος: Η απάντηση σε μορφή string.

{% if interpreter.slug == "js-node" %}

fizzBuzz(6) == "Fizz"
fizzBuzz(5) == "Buzz"
fizzBuzz(7) == "7"
{% else %}
checkio(15) == "Fizz Buzz"
checkio(6) == "Fizz"
checkio(5) == "Buzz"
checkio(7) == "7"
{% endif %}

Πώς χρησιμοποιείται: Εδώ θα μάθεις πώς να γράφεις μια πολύ απλή συνάρτηση και θα δουλέψεις με την εντολή if-else.

Προϋποθέσεις: 0 < number ≤ 1000
