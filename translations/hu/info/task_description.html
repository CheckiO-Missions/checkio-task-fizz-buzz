A „Fizz buzz” egy szójáték, melynek segítségével megfogjuk tanítani robotjainkat az osztás műveletére. Gyerünk és okítsuk a számítógépet!

Egy függvényt kell írni itt, amely egy pozitív számot kap bemenetként és visszaadja a következő kifejezést:
"Fizz Buzz", ha a szám osztható 3-ra és 5-re;
"Fizz", ha a szám osztható 3-ra;
"Buzz", ha a szám osztható 5-re;
Egy számot, mint egy szöveget az összes többi esetre.

Bemeneti adatok: egy egész szám (int).

Kimeneti adatok: a felelet karakterláncként (str).

Példák: 

{% if interpreter.slug == "js-node" %}
fizzBuzz(15) == "Fizz Buzz"
fizzBuzz(6) == "Fizz"
fizzBuzz(5) == "Buzz"
fizzBuzz(7) == "7"
{% else %}
checkio(15) == "Fizz Buzz"
checkio(6) == "Fizz"
checkio(5) == "Buzz"
checkio(7) == "7"
{% endif %}

Alkalmazása: Ennek a feladatnak a segítségével el lehet sajátítani egy egyszerűbb függvény deklarálásának módját és a ha-akkor-különben (if-else) feltételes utasítás használatát.

Előfeltételek:
0 < number ≤ 1000
