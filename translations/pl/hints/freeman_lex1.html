Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

jest to prosta misja z prostym pomysłem na rozwiązanie: proszę utworzyć sprawdzanie wielu warunków i zwrócić odpowiednią wartość dla każdego przypadku.

Ok, od czego powinienem zacząć?

Przede wszystkim proszę poczytać o if-elif-else statement i używać go w swoim kodzie: 

if ...:
    ...
elif ...:
    ...
else:
    ...

Ok. Co robić dalej?

Mogą Państwo sprawdzić, jak liczba jest podzielna przez inną za pomocą % (modulo) operator. 

if num % n == 0: # num is evenly divisible by n
    ...
elif num % n1 == 0:
    ...
else:
    ...

Bardziej pythonicznym sposobem jest zapisanie jako: 

if not num % n:
    ...
elif not num % n1:
    ...
else:
    ...

Ok, jakich liczb powinienem użyć zamiast n, n1?

Proszę sprawdzić podzielność od konkretnej do ogólnej. Ponieważ mniej liczb jest parzyście podzielnych przez 15, a następnie przez 5 lub 3, ten przypadek jest bardziej szczegółowy, więc to sprawdzenie powinno być pierwsze. 5 nie jest parzyście podzielny przez 3, więc dalsza kolejność nie jest ważna.

Proszę o małą podpowiedź!

Oto pański kod: 

if not num % 15:
    ...
elif not num % 5:
    ...
elif not num % 3:
    ...
else:  
    ...

Jak uzupełnić warunki?

Proszę dodać poprawny zwrot do odpowiednich przypadków: ciągów "Fizz", "Buzz" i "Fizz Buzz".

W drugim przypadku (gdy num nie jest podzielne przez 15, 5, 3 parzyście), funkcja powinna zwrócić num jako ciąg znaków. Jeśli potrzebują Państwo łańcuchowej reprezentacji liczby, proszę użyć wbudowanej funkcji str().

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę opakować swój kod w wyrażenie funkcyjne.

Oto całe rozwiązanie:

def checkio(num: int) -> str:

    if not num % 15:
        return "Fizz Buzz"
    elif not num % 5:
        return "Buzz"
    elif not num % 3:
        return "Fizz"
    else:  
        return str(num)

Przy okazji, jeśli Państwa przypadki warunkowe kończą się instrukcją return, zazwyczaj nie potrzebują Państwo elif i else - wystarczy if: 

def checkio(num: int) -> str:

    if not num % 15:
        return "Fizz Buzz"
    if not num % 5:
        return "Buzz"
    if not num % 3:
        return "Fizz"
    return str(num)

Jasne, proszę łapać!

Ostateczna odpowiedź jest zbudowana z części: jeśli ciąg zostanie pomnożony przez 0, stanie się pustym ciągiem "". 

def checkio(n: int) -> str:

    return 'Fizz'*(not n%3)+' '*(not n%15)+'Buzz'*(not n%5) or str(n)

W tym rozwiązaniu unikamy dzielenia przez 15, ale w przypadku, gdy num jest podzielny zarówno przez 5, jak i 3, oznacza to, że jest on również podzielny przez 15. Tak więc oba ciągi są obecne na liście. 

def checkio(num: int) -> str:
    
    res = []
    if not num % 3:
        res.append('Fizz')
    if not num % 5:
        res.append('Buzz')
    res = ' '.join(res) or str(num)

    return res
