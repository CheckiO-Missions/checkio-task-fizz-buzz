"Fizz buzz" é um jogo com palavras que iremos utilizar para ensinar os robôs sobre divisão. Vamos aprender sobre computadores.

Você deve escrever uma função a qual recebe um inteiro positivo e retorna:
"Fizz Buzz" se o número for divisível por 3 e por 5;
"Fizz" se o número for divisível por 3;
"Buzz" se o número for divisível por 5;
O número como uma palavra (string) para todos os outros casos.

Entrada: Um número como um inteiro (int).

Saida: A resposta como uma palavra (string).

Exemplos: 

checkio(15) == "Fizz Buzz"
checkio(6) == "Fizz"
checkio(5) == "Buzz"
checkio(7) == "7"

Como isto é utilizado: Aqui você podera aprender como escrever uma das funções mais simples e trabalhar com if-else.
