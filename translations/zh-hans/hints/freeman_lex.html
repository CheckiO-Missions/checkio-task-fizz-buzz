def checkio(num: int) -> str:

    if not num % 15:
        return "Fizz Buzz"
    elif not num % 5:
        return "Buzz"
    elif not num % 3:

最终答案是由几个部分组成的：如果字符串乘以0 ，就会变成空字符串"" 。 

def checkio(n: int) -> str:

    return 'Fizz'*(not n%3)+' '*(not n%15)+'Buzz'*(not n%5) or str(n)

在这个方案中，避免了被15 除，但如果num 同时被5 和3 平均整除，则意味着它也能被15 平均整除。因此这两个字符串都出现在列表中。 

def checkio(num: int) -> str:
    
    res = []
    if not num % 3:
        res.append('Fizz')
    if not num % 5:
        res.append('Buzz')
    res = ' '.join(res) or str(num)

    return res
