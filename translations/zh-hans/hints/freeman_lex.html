最后的答案是由部分组成的：如果字符串乘以0 ，就会变成一个空字符串"" 。 

def checkio(n: int) -> str:

    return 'Fizz'*(not n%3)+' '*(not n%15)+'Buzz'*(not n%5) or str(n)

在这个解决方案中，可以避免除以15 ，但是如果num 同时被5 和3 匀称地除以，这意味着它也可以被15 匀称地除以。所以这两个字符串都出现在列表中。 

def checkio(num: int) -> str:
    
    res = []
    if not num % 3:
        res.append('Fizz')
    if not num % 5:
        res.append('Buzz')
    res = ' '.join(res) or str(num)

    return res
