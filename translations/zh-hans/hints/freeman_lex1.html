如何完成条件？

你需要在各自的情况下添加正确的返回：字符串"Fizz","Buzz" 和"Fizz Buzz" 。

在其他情况下（当num 不能被15, 5, 3 均分时），该函数应该返回num 作为字符串。如果你需要一个数字的字符串表示，请使用str() 内置函数。

那么，我的最终解决方案应该是什么样子的呢？

只需将你的代码包装成函数表达式。

下面是整个解决方案：

def checkio(num: int) -> str:

    if not num % 15:
        return "Fizz Buzz"
    elif not num % 5:
        return "Buzz"
    elif not num % 3:
        return "Fizz"
    else:  
        return str(num)

顺便说一下，如果你的条件案例以return 语句结束，你通常不需要elif 和else -if 就够了：

最后的答案是由部分组成的：如果字符串乘以0 ，就会变成一个空字符串"" 。 

def checkio(n: int) -> str:

    return 'Fizz'*(not n%3)+' '*(not n%15)+'Buzz'*(not n%5) or str(n)

在这个解决方案中，可以避免除以15 ，但是如果num 同时被5 和3 匀称地除以，这意味着它也可以被15 匀称地除以。所以这两个字符串都出现在列表中。 

def checkio(num: int) -> str:
    
    res = []
    if not num % 3:
        res.append('Fizz')
    if not num % 5:
        res.append('Buzz')
    res = ' '.join(res) or str(num)

    return res
